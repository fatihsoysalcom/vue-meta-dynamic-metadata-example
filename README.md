# Vue Meta Dynamic Metadata Example

This example demonstrates how to use the `vue-meta` library in a Vue.js Single Page Application (SPA) to dynamically manage metadata. It shows how to define global metadata for the entire app and how to specify page-specific titles and meta tags for different components, which are crucial for SEO and social media sharing.

## Language

`html`

## How to Run

1. Save the code as `index.html`.
2. Open `index.html` in your web browser.
3. Click the navigation buttons to switch between pages and observe the browser tab title and the `<head>` section (using browser developer tools) updating dynamically.
